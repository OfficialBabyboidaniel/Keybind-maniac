Keybinds - Linux
Open a new terminal session : ctrl + alt + t 
Swap between workspaces : ctrl + alt + leftArrow/ rightArrow 

While in terminal -> 
Close terminal session : ctrl + d
Copy from terminal : ctrl + shift + c 
Paste into terminal : ctrl + shift + v  
Clear terminal : ctrl + l 
Go to begining of line : ctrl + a  ( maybe change to match vscode ?????)
Go to end of line : ctrl + e (maybe change to match vs code ?????)
Go back one character : ctrl + b
Go back one block : alt + b 
Go forward one character : ctrl + f
Go forward one block : alt + f 
Toggle beginning / cursor location : ctrl + xx (press twice)
Delete char under cursor : ctrl + d 
Delete word after cursor : alt + d 
swap position of two blocks : alt + t ( takes block left of cursor and switches with cursor's block -> will contunie to switch that block forward with upcoming words if command reapeated )
